# **Birmarket – Test Strategy**

## 1. Məqsəd (Test Objectives)  

* Birmarket platformasının **funksional, stabil, təhlükəsiz və istifadəçi-dostu** olmasını təmin etmək
* Production-a çıxmazdan əvvəl **kritik riskləri minimuma endirmək**
* Biznes axınlarının (satış, ödəniş, sifariş) düzgün işlədiyini təs  diqləmək

---

## 2. Əhatə dairəsi (Scope)

### **In-Scope**

* İstifadəçi qeydiyyatı və login
* Məhsul kataloqu və axtarış
* Səbət (Cart) və Checkout
* Ödəniş prosesi
* Sifarişlərin idarəsi
* Profil və adreslər
* Kampaniyalar / endirimlər
* Admin panel (məhsul, qiymət, stok)
* Bildirişlər (email / push – varsa)

### **Out-of-Scope**

* 3rd party sistemlərin daxili testləri
